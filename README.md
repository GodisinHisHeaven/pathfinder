### In order to test the program, you need to run the following command on the EWS:

    mkdir obj
    mkdir bin
    module load llvm/6.0.1
    make
    bin/tests

### For customized output, you can run the following command on the EWS:

    make exec
    bin/exec [start node] [end node]
